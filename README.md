#  Python CLI To-Do List

A simple and intuitive Command Line Interface (CLI) application to manage daily tasks. 

This project was built as part of my Software Engineering journey to practice Python fundamentals, file handling (JSON), error treatment, and CRUD (Create, Read, Update, Delete) operations.

##  Features
- **Add tasks:** Create new tasks easily.
- **List tasks:** View all your tasks and their current status (Pending / Completed).
- **Toggle status:** Switch a task's status from Pending to Completed (and vice versa).
- **Remove tasks:** Delete tasks you no longer need via their unique ID.
- **Data Persistence:** All data is safely stored in a `data.json` file, ensuring no tasks are lost when the application is closed.
- **Input Validation:** Error handling (`try/except`) implemented to prevent the program from crashing if invalid inputs (e.g., letters instead of numbers) are provided.

##  Technologies Used
- **Python 3**
- **JSON** (Data storage)
- **OS Module** (Cross-platform terminal clearing)
